# Prism

AI-powered knowledge management for technical teams.

Prism connects to your AI assistant and gives it memory — your specs, decisions, 
architecture, and roadmap. You talk to Claude. Prism makes Claude smarter about 
your projects.

## Getting Started

1. Download the latest release from the [Releases](../../releases) page
2. Follow the setup instructions included with your release
3. Connect Prism to your Claude instance
